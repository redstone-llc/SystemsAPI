# SystemsAPI

> [!WARNING]
> This library is a work in progress! Expect constant changes at this stage.

SystemsAPI is a fabric library that abstracts Hypixel Housing's systems for other mods to use. For an example implementation, see [HTSL Reborn](https://github.com/sinender/HTSLReborn).

## Features:
- [x] Action Importing
- [x] Action Exporting
- [ ] Optimized Action Importing (optimally keeps/changes existing actions to make imports quick!)
